Стабильная электросеть

Вова использует очень много электроники, а, как известно, каждый уважающий себя электро– прибор требует подключения к электричеству.


Для подключения всех своих устройств он использует ﻿nn﻿ сетевых фильтров. В электросеть города подключен главный сетевой фильтр с номером 1. Все остальные фильтры подключены к главному напрямую или через несколько промежуточных фильтров. Вова соблюдает технику безопасности, поэтому фильтры не образуют циклов и не подключаются сами к себе. Иными словами, подключение фильтров образует дерево.


Про каждый фильтры известна специальная характеристика ﻿aiai​﻿. Также про все фильтры, кроме первого, известно ﻿lili​﻿ — длина провода, которым он подключен к следующему фильтру.


Будем говорить, что фильтр ﻿ii﻿ нагружает фильтр ﻿j  (i≠j)j(i​=j)﻿, если ﻿ii﻿ прямо или через несколько промежуточных фильтров подключен к электросети через фильтр ﻿jj﻿ и суммарная длина проводов между фильтрами ﻿length(i,j)≤ailength(i,j)≤ai​﻿.


Для каждого фильтра в сети Вовы найдите количество фильтров, которые его нагружают.


Формат входных данных

В первой строке входных данных дано одно число ﻿nn﻿ — количество фильтров в сети Вовы с учетом главного ﻿(1≤n≤105)(1≤n≤105)﻿.


Во второй строке входных данных дано ﻿nn﻿ целых чисел ﻿a1,…,ana1​,…,an​﻿ — характеристика фильтров ﻿(1≤ai≤109)(1≤ai​≤109)﻿.


Далее следует ﻿n−1n−1﻿ целое число ﻿p2,…,pnp2​,…,pn​﻿ — номер фильтра, к которому подключен соответ- ствующий фильтр ﻿(1≤pi≤n)(1≤pi​≤n)﻿.


В последней строке ﻿n−1n−1﻿ целое число ﻿l2,…,lnl2​,…,ln​﻿ — длина провода соответствующего фильтра ﻿(1≤li≤109)(1≤li​≤109)﻿.


Формат выходных данных

Выведите в одной строке n чисел, ﻿ii﻿-е из которых должно быть равно количеству фильтров, которые нагружают фильтр с номером ﻿ii﻿.
