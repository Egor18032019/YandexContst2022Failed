Для некоторых терминов с большим количество букв принято использовать сокращения: l10nl10nl10n вместо localizationlocalizationlocalization или i18ni18ni18n вместо internationalizationinternationalizationinternationalization.

Вам дан набор из nnn строк длиной не более 20 символов.

Для каждой строки www определим сокращение pNspNspNs, где ppp – некоторый непустой префикс строки www, sss – некоторый непустой суффикс строки www, NNN – целое число больше единицы, которое задает количество пропущенных букв между префиксом и суффиксом. Будем рассматривать только такие сокращения, где длины ppp и sss совпадают.

Из всех сокращений для каждого слова из набора найдите такое, что никакое другое слово не может быть сокращено таким же образом.
Формат ввода

В первой строке записаны число nnn (1≤n≤50 0001 \le n \le 50\,0001≤n≤50000).

В каждой из следующих nnn строк записано одно слово www (4≤∣w∣≤204 \le |w| \le 204≤∣w∣≤20). Все слова различны и состоят только маленьких букв английского алфавита.
Формат вывода

Выведите nnn строк, по одной строке для каждого слова из входных данных (в порядке следования во входных данных) – минимальное по длине подходящее под условие задачи сокращение, если подходящего сокращения нет, выведите слово без сокращения.
