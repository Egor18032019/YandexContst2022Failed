Расстояние Хэмминга (кодовое расстояние) — число позиций, в которых соответствующие символы двух слов одинаковой длины различны. В более общем случае расстояние Хэмминга применяется для строк одинаковой длины любых qqq-ичных алфавитов и служит метрикой различия (функцией, определяющей расстояние в метрическом пространстве) объектов одинаковой размерности.

Вам даны пары бинарных строк одинаковой длины (sss, ddd). Найдите бинарную строку ttt, для которой величина max⁡(hamming(s,t),hamming(d,t))\max(hamming(s, t), hamming(d, t))max(hamming(s,t),hamming(d,t)) минимальна (hamming(s,t)hamming(s, t)hamming(s,t) — расстояние Хемминга между строками sss и ttt).

Если бинарных строк минимизирующих данную величину несколько, выведите любую из них.
Формат ввода

В первой строке записаны два целых числа nnn и qqq (1≤n≤251 \le n \le 251≤n≤25, 1≤q≤100 0001 \le q \le 100\,0001≤q≤100000) — длина строк в наборе тестовых данных и количество тестовых пар.

В каждой из следующих qqq строк записаны две бинарные строки sss и ttt длины nnn. Строки разделены одиночным пробелом.
Формат вывода

Для каждой из qqq тестовых пар выведите найденную строку ttt. Тестовые пары нужно обрабатывать в порядке следования во входных данных.
